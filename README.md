# Retos_RPeM_Analysis
Datos y análisis del estudio RPeM-TTO en atletas universitarios. El RPeM es un índice de eficiencia mecánica concéntrica con unidades de velocidad, calculado a partir de saltos verticales con apps móviles. Funciona como un indicador práctico para detectar reducciones tempranas en el rendimiento y la preparación neuromuscular del atleta (readiness).
